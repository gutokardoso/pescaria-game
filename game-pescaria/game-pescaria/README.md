# Game de Pescaria

Projeto estático em HTML, CSS e JavaScript, pronto para publicar no GitHub e na Vercel.

## Como publicar na Vercel

1. Suba todos os arquivos deste ZIP para um repositório no GitHub.
2. Na Vercel, importe o repositório.
3. Framework Preset: `Other`.
4. Build Command: deixe vazio.
5. Output Directory: deixe vazio ou use `.`.

## Estrutura

- `index.html`: game completo.
- `assets/`: imagens do barco, céu, fundo do mar e peixinhos.
